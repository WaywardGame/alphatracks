# Alpha Tracks
Adds two tracks to the game. One used earlier in production (during the early alphas) entitled "pixplz" by Oddmund Str√∏mme, and another previously unreleased track entitled "The Highlands" by Will Phillips.

## Cloning/Setup:
```
git clone https://github.com/WaywardGame/alphatracks.git
cd alphatracks
On Windows: path/to/wayward/install/wayward.cmd +mod update .
On macOS: path/to/wayward/install/Wayward.app/Contents/MacOS/Electron +mod update .
On Linux: path/to/wayward/install/wayward +mod update .
```

## Building:
Open the folder in Visual Studio Code and build with Ctrl+Shift+B.
